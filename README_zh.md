vase是全新的移动端跨平台UI开发框架，一份代码同时编译运行在Android，iOS，Web浏览器，桌面端。

在Android上编译为java字节码，在iOS上编译为C，在Web上编译为Javascript。

### 特性
提供下列特性，让开发者能容易地创建原生编译的好看的app。

- 声明式编程。序列化格式是代码语法的子集，UI描述可以用配置文件也可以用代码。
- 丰富的UI控件。
- 简单强大的布局容器。
- 灵活的样式系统。
- 吸引人的微动画。
- 基于async/awit的Http客户端。
